# Yield Optimization

Maximize your DeFi yields across multiple protocols and chains with ease.

Kixago's Yield Optimization feature aggregates lending opportunities across Aave, Compound, MakerDAO, and other protocols, normalizing the data so you can make informed decisions quickly.

## How It Works

1. Query all supported protocols for a specific token.
2. Normalize APY and reward data for easy comparison.
3. Rank opportunities by real APY after accounting for gas, rewards, and slippage.
4. Filter by chain, risk, or other criteria to suit your strategy.

## Quick Start Example

```javascript
import { fetchLendingOpportunities } from '@kixago/sdk';

const token = 'USDC';

async function getBestOpportunities() {
  const opportunities = await fetchLendingOpportunities(token);

  // Sort by real APY
  opportunities.sort((a, b) => b.apy_real - a.apy_real);

  console.log('Top 3 lending opportunities for', token);
  console.log(opportunities.slice(0, 3));
}

getBestOpportunities();
```

## Features

| Feature               | Description                                                       |
| --------------------- | ----------------------------------------------------------------- |
| Multi-Protocol        | Aggregate opportunities across Aave, Compound, MakerDAO, and more |
| Multi-Chain           | Ethereum, Polygon, Arbitrum, Base support                         |
| Real APY Calculation  | Accounts for base APY, reward tokens, and estimated gas           |
| Risk Assessment       | Risk scores help avoid volatile or unsafe markets                 |
| Sortable & Filterable | Sort by APY, TVL, risk, or chain                                  |
